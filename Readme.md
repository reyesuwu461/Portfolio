
# Samuel Reyes - Full Stack Developer Portfolio

![Portfolio Screenshot](img/Captura%20de%20pantalla%202025-08-30%20150747.png)

## Overview

This is my personal portfolio as a full stack developer, where I showcase my skills, projects, and experience. The site is fully responsive, with modern animations, light/dark mode, interactive effects, and a retro game section.

## Features

- **Modern & Responsive Design**: Adapts to any device (desktop, tablet, mobile).
- **Light/Dark Mode**: Switch the visual theme with an animated toggle.
- **Animations & Effects**: Smooth transitions, floating particles, interactive cards, and typewriter effect in the title.
- **Projects Section**: Showcases 3 featured projects and an interactive loading animation example.
- **Retro Game**: Includes the "Pandemoniun Crusader" game made with JavaScript, HTML, and CSS.
- **Contact Form**: Allows visitors to send me messages directly.
- **Social Links**: Access to my networks and GitHub.

## Technologies Used

- **Frontend**:
  - Semantic HTML5
  - CSS3 (Custom Properties, Flexbox, Grid, animations)
  - JavaScript (interactivity, animations, dark mode, particles)
- **Other**:
  - SVG for animated graphics
  - jQuery (for the loading animation)
